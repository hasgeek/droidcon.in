Beginner's Android Workshop

23 - 24 November 2013

A hands-on introduction to android app development mainly focused on newbies.

  SCHEDULE
DAY 1
Android overview and history
  • Evolution
  • Feature Set
  • Versions
Android stack
  • Android architecture
  • Tools
Main android building blocks
  • Overview of Activities, Services, Broadcast Receivers, Content Provider
Simple hello world program
  • Setup IDE - Device Manager, Emulator configuration
  • Android Project structure
  • Executing and debugging application on the emulator and device
Constituents of Application Manifest file
Building a simple functional Android Application
  • Laying out the screen with UI Widgets
  • Handling events
Lunch break
Processes and Threads
  • Application process and main thread
  • Foreground Process
  • Visible Process
  • Services
  • Background Process
  • Empty Process
Activity and Service Life cycle in detail
Overview of Intents
  • Activity calling another activity
  • Sample application using intents
Developing a music player application
  • Overview of Media Player APIs
  • Accessing Media Player from a foreground activity
  • Moving media player access to a Service
DAY 2
Asynchronous Processing
  • Async task
  • Intent Services
Broadcast Receiver
Notifications and Pending intent
  • Moving Services to foreground mode
  • Using Pending Intent
  • Publish notification
Google Cloud Messaging
Lunch break
Shared Preferences
Accessing files
Databases and content providers
Prototyping, Creating and Publishing Android app on Google Play store
Demonstrate accessing device APIs like Camera, Accelerometer

Intermediate/Advanced Android Workshops

26 - 27 November 2013

Two-day intensive hands-on workshops that dives deep into the advanced techniques of Android development.

  SCHEDULE
Time Workshop
26th Nov: 9AM - 11AM OpenCV based app development on Android Devices
26th Nov: 11.30AM - 1.30PM OpenGLES Graphics Programming in Android
26th Nov: 3PM - 5PM Building your own ROMS
27th Nov: 9AM - 11AM Android Build, Test and Deploy with Gradle
27th Nov: 11.30AM - 1.30PM Unit Test your Android application using Robolectric
      INSTRUCTORS

  • Hiemanshu Sharma

    Co Founder at Mobule

    @hiemanshu

    Building your own ROMs

    This is a workshop that will help you with building ROMs for the devices that are currently supported by CyanogenMod.
    It will cover details on getting the source code (I will have a recent copy of it with me so we dont waste bandwidth), setting up your machine to build ROMs, and how to do the most common tasks and debugging that go on with building ROMs.

    For prerequisites and getting in touch with instructor, visit Funnel

  • Ravindra Kumar

    Front End Engineer at Cleartrip.com

    @ravidsrk

    Android Build, Test and Deploy with Gradle

    The following topics are included in this workshop:
    - Unit testing in the Dalvik Virtual Machine
    - Testing with Spoon and Robotium
    - Taking advantage of Android Logging
    - UI testing with Spoon
    - Integrating the build environment with Gradle

    For prerequisites and getting in touch with instructor, visit Funnel

  • Leena S N

    Head of Engineering at Multunus

    @leenasn

    Unit Test your Android application using Robolectric

    This workshop is for giving an overview on how to write faster, maintainable and behavioural tests using Robolectric, an Android Unit Testing Framework. At the end of the session, the attendees should be able to:
    - Learn about basic Android Unit Testing and understand the issues with the default AndroidTestRunner
    - Overview of Robolectric
    - How to write tests using Robolectric and run it in JVM [i.e. instead of Dalvik]
    - How to integrate Robolectric tests with your Continuous Integration Server such as Jenkins

    For prerequisites and getting in touch with instructor, visit Funnel

  • Wingston Sharon

    Developer at Premiere Digital Services

    @wingston

    OpenCV based app development on Android Devices

    This workshop would focus on introducing basic image processing with the openCV image processing framework in Android. Participants can learn how to set up a complete environment for OpenCV development in Eclipse and practically try out some image processing examples during the practical workshop on their android device. Participants can learn how to Integrate openCV for image processing real time video streams from the camera of any android device for their custom applications.

    For prerequisites and getting in touch with instructor, visit Funnel

  • Arvind Devaraj

    Android Professional and Technical Lead at NalandaU virtual university

    @arvind_devaraj

    OpenGLES Graphics Programming in Android

    The objective of this workshop is to get hands on experience in developing Graphics programs in Android. OpenGL is a 3D graphics API for rendering. Knowing the basics of openGLES will help to create better graphics programs . Also you can improve performance of the programs by utilizing the GPU

    For prerequisites and getting in touch with instructor, visit Funnel
